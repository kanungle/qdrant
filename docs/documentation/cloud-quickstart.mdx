## Setup a Qdrant Cloud Cluster

<Steps>
<Step> Register for a [Cloud account]("") with your email, Google or Github credentials.</Step>
<Step>
Go to Clusters and follow the onboarding instructions under Create First Cluster.
<Frame>
  <img src="/images/create-cluster.png" />
</Frame>
</Step>

<Step>
  When you create it, you will receive an API key. You will need to copy it and
  store it somewhere self. It will not be displayed again. If you loose it, you
  can always create a new one on the Cluster Detail Page later.
  <Frame>
    <img src="/images/api-key.png" />
  </Frame>
</Step>

</Steps>

## Access the Cluster UI

<Steps>
<Step> Click on Cluster UI on the Cluster Detail Page to access the cluster UI dashboard.</Step>
<Step>
Paste your new API key here. You can revoke and create new API keys in the API Keys tab on your Cluster Detail Page.
</Step>

<Step>
  The key will grant you access to your Qdrant instance. Now you can see the
  cluster Dashboard.
  <Frame>
    <img src="/images/access-dashboard.png" />
  </Frame>
</Step>

</Steps>

## Authenticate via SDKs

Now that you have your cluster and key, you can use our official SDKs to access Qdrant Cloud from within your application.

<Tabs>
  <Tab
    title="bash"
    icon="code"
  >
  ```
curl \
  -X GET https://xyz-example.eu-central.aws.cloud.qdrant.io:6333 \
  --header 'api-key: <your-api-key>'

# Alternatively, you can use the `Authorization` header with the `Bearer` prefix

curl \
 -X GET https://xyz-example.eu-central.aws.cloud.qdrant.io:6333 \
 --header 'Authorization: Bearer <your-api-key>'

```
</Tab>
<Tab
  title="python"
  icon="python"
>
```

from qdrant_client import QdrantClient

qdrant_client = QdrantClient(
host="xyz-example.eu-central.aws.cloud.qdrant.io",
api_key="<your-api-key>",
)

```
</Tab>

  title="typescript"

```
const client = new QdrantClient({

```
</Tab>
<Tab title="rust" icon="rust">

```
use qdrant_client::Qdrant;

let client = Qdrant::from_url("https://xyz-example.eu-central.aws.cloud.qdrant.io:6334")

```
</Tab>
<Tab
  title="java"


## Try the Tutorial Sandbox

<Steps>
<Step> Open the interactive Tutorial. Here, you can test basic Qdrant API requests.</Step>
<Step>
Using the Quickstart instructions, create a collection, add vectors and run a search.
</Step>

<Step>
The output on the right will show you some basic semantic search results.
<Frame>
  <img src="/images/interactive-tutorial.png" />
</Frame>
</Step>

</Steps>
## That’s Vector Search!

When ready, use the Console and our complete REST API to try other operations.
